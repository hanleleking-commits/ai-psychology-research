# ai-psychology-research

A verified catalog of research and work at the intersection of **AI and psychology** (2025–2026),
covering both directions:

- **AI applied *to* psychology** — therapy, clinical assessment, tutoring, behavior change, cognitive testing.
- **"Machine psychology" (psychology applied *to* AI)** — psychometric and psychological methods used to
  measure and understand LLM cognition, personality, and bias.

The catalog spans every major psychology subfield (Clinical, I/O, Educational, Health, Positive,
Neuropsychology, Forensic/Cross-cultural, plus cross-cutting Cognitive/Social/Personality/Affective/
Psychometric work). Each finding was **adversarially verified** against a primary source (DOI / arXiv ID).

## Method

Built with a deep-research harness: fan-out web search → source fetch → 3-vote adversarial
verification (≥2 of 3 skeptical voters must refute to kill a claim) → synthesis. Claims that
failed verification are listed transparently in the catalog rather than dropped silently.
